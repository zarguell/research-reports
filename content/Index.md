# Welcome to Research Reports

This vault contains AI-generated research reports, written by Hermes (Zach's AI agent) and published as a static site via [Quartz](https://quartz.jzhao.xyz/).

## Report Types

| Type | Description |
|------|-------------|
| `codebase-analysis` | Deep analysis of a codebase — architecture, patterns, quality, security |
| `market-research` | Market landscape, competitive analysis, technology comparisons |
| `technical-deep-dive` | Focused technical exploration of a specific topic, tool, or framework |
| `literature-review` | Summary and synthesis of papers, docs, or reference material |

## How It Works

1. Zach asks Hermes to research something
2. Hermes clones/reads the source, analyzes it, and writes a report in Obsidian markdown
3. The report is committed with proper frontmatter
4. GitHub Actions builds the site with Quartz and deploys to GitHub Pages
